Avatars are use for profile pictue, you can add (.avatar) class to use them

Profile Avatar

Different Sized Avatar

For using diffrent sizes of avatar you can add (.smallavt) for small avatar and (.mediumavt), (.largeavt) for medium, large avatar.

Letter Avatar

In this type of avatar (.primary, .secondary, .warning) are primary, secondary, warning background color. For centering letter (.center) class is used which justify content to center and align item to center.
