Button

Buttons allow user to do action and also called as call to action. We have wide range of buttons and they are below.

Solid Button

Solid button are created using (.solid-btn) class and colors are given by (.primary, .secondary, .success, .warning).

Outline Button

Outline button are created using (.outline-colorname)

link Button

Link button are created by using (.btn-link) class.

Icon Button

In this type of button icon is added using icon link.

Floating Action Button

Floating action button is created using (.floating-btn) class and it is inside (box-floatingbtn) class.
